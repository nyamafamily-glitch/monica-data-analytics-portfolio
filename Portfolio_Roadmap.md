# Data Analytics Portfolio Roadmap

This roadmap outlines the steps for building five data analytics projects to showcase skills in visualization, cloud technologies, and ETL pipelines.

---

## 1. Power BI Sales Dashboard
**Goal:** Create an interactive dashboard analyzing sales performance.

### Dataset
- [Global Superstore Dataset](https://www.kaggle.com/datasets/rohitsahoo/sales-data)
- Or [Microsoft Sample Sales Data](https://learn.microsoft.com/en-us/power-bi/sample-datasets)

### Tools
- Power BI, Power Query, DAX

### Steps
1. Import dataset into Power BI.
2. Clean data using Power Query (remove duplicates, handle missing values).
3. Create calculated columns (e.g., Profit Margin).
4. Build visuals:
   - KPIs: Total Sales, Profit, Quantity.
   - Sales by Region (Map or Bar Chart).
   - Sales by Category (Pie Chart).
   - Monthly Sales Trend (Line Chart).
5. Add slicers for Region, Category, Year.
6. Save `.pbix` file and take screenshots.

### Deliverables
- Power BI dashboard file.
- Screenshots.

## 2. Python Customer Segmentation
**Goal:** Segment customers using clustering techniques.

### Dataset
- [Mall Customer Segmentation Data](https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python)

### Tools
- Python (pandas, scikit-learn), MySQL

### Steps
1. Load dataset in Jupyter Notebook.
2. Perform data cleaning and preprocessing.
3. Apply K-Means clustering.
4. Visualize clusters using matplotlib/seaborn.
5. Store results in MySQL database.

### Deliverables
- Jupyter Notebook (`.ipynb`).
- SQL scripts.
- Cluster visualization screenshots.

## 3. Google Analytics Web Traffic
**Goal:** Analyze website traffic and engagement.

### Dataset
- Use Google Analytics demo account.

### Tools
- Google Analytics, Excel/Power BI for visualization.

### Steps
1. Access demo account and export data.
2. Analyze traffic sources, bounce rate, session duration.
3. Create charts for top-performing pages and traffic trends.

### Deliverables
- Analysis report.
- Visualizations.

---

## 4. Snowflake Data Warehouse
**Goal:** Build and query a cloud data warehouse.

### Dataset
- Public dataset (e.g., sales or customer data).

### Tools
- Snowflake, SQL

### Steps
1. Create Snowflake account.
2. Load dataset into Snowflake.
3. Write SQL queries for insights (e.g., top-selling products).
4. Connect Snowflake to BI tool for visualization.

### Deliverables
- SQL scripts.
- Screenshots of queries and dashboards.

---

## 5. Matillion ETL Pipeline
**Goal:** Build ETL pipeline for data integration.

### Dataset
- Same dataset used in Snowflake project.

### Tools
- Matillion, Snowflake

### Steps
1. Set up Matillion ETL environment.
2. Create pipeline to extract CSV data.
3. Transform data (cleaning, aggregations).
4. Load data into Snowflake.

### Deliverables
- Pipeline documentation.
- Screenshots of ETL process.
